vue-router文档 npm package

router.replace(path)

和 router.go(path) 类似，但是并不会在浏览器历史创建一条新的纪录。

参数

  • path: String

    此路径为一个普通路径（也就是说没有动态片段或者全匹配片段）。路径不能以 / 开头，会以相对于当前路径的方式进行解析。
